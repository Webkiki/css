Ce este Git?

Git este un sistem popular de control al versiunilor.

Este folosit pentru:

A urmari modificarile codului, a urmari cine a facut modificarile, colaborare la codificare

Ce face Git?

Va permite sa gestionati proiecte cu depozite, cloneaza un proiect pentru a lucra pe o copie locala, puteti controla si urmari modificarile cu Staging and Committing Branch si Merge pentru a permite lucru la diferite parti si versiuni ale unui proiect.

Pentru a folosi Git - vom deschide un terminal (Command Shell). Apoi vom verifica daca Git este instalat si daca este instalat corect, folosim comanda git --version in terminal.

Definim userul si mail-ul contului nostru de GitHub. Pentru asta vom rula 2 comenzi:git config --global user.name "USERUL_TAU" si git config --global user.email "EMAILUL_TAU"

Aceste comenzi se utilizeaza exclusiv la prima configurare de GIT.

Primul repository

Pentru a lega folderul local de repository-ul din github trebuie sa deschidem browserul si se copiaza link-ul generat de github.

Apoi, in terminal, introducem comanda git spațiu clone spațiu și link-ul pe care l-am copiat.

Pentru a executa comenzi trebuie sa navigam in folderul pe care vrem sa il incarcam. Aici folosim comanda cd spatiu si "calea" catre folderul in care vrem sa ajungem.

Pentru a verifica folderul in care ne aflam folosim comanda ls sau ls -lah pentru mai multe detalii.

Pentru a incarca fisierele de pe calculator pe github avem nevoie de 3 comenzi: git add ., git commit -m "feat: fisier nou de index" (m contine mesajul pe care vrem sa il trimitem), git push.
