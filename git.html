Ce este Git?

Git este un sistem popular de control al versiunilor.

Este folosit pentru:

A urmari modificarile codului, a urmari cine a facut modificarile, colaborare la codificare

Ce face Git?

Va permite sa gestionati proiecte cu depozite, cloneaza un proiect pentru a lucra pe o copie locala, puteti controla si urmari modificarile cu Staging and Committing Branch si Merge pentru a permite lucru la diferite parti si versiuni ale unui proiect.

Pentru a folosi Git - vom deschide un terminal (Command Shell). Apoi vom verifica daca Git este instalat corect.
