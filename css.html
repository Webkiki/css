Proprietati CSS

Font, text

Font-family - este de fapt o lista de fonturi din care browserul va folosi in ordinea in care le recunoaste (primul folosit va fi primul din lista, daca nu este recunoscut il foloseste pe al doilea si tot asa mai departe)

Font-style - proprietatea font-style este folosită mai ales pentru a specifica textul italic. Aceasta proprietate are 3 valori

Text-align - proprietatea text-align este utilizată pentru a seta alinierea orizontală a unui text. Un text poate fi aliniat la stânga sau la dreapta, centrat sau justificat.

text-decoration - proprietatea text-decoration-line este folosită pentru a adăuga o linie de decor textului. Se pot combina mai multe valori.

font-size - proprietatea font-size stabilește dimensiunea textului. Valoarea mărimii fontului poate fi o dimensiune absolută sau relativă.

color, background

background-color - proprietatea background-color specifică culoarea de fundal a unui element.

color - proprietatea color poate fi atribuita: background ului, textului, etc

background-image - roprietatea background-image specifică o imagine de utilizat ca fundal al unui element. În mod implicit, imaginea se repetă astfel încât să acopere întregul element.

link-uri

  • a: link - un link normal, nevizitat
  • a: visited - un link pe care utilizatorul l-a vizitat
  • a: hover - un link atunci când utilizatorul trece mouse-ul peste el
  • a: active - un link în momentul în care se face clic pe acesta

dimensiuni, border, margin, padding

width/height - proprietățile CSS înălțime și lățime sunt utilizate pentru a seta înălțimea și lățimea unui element. proprietatea CSS max-width este utilizată pentru a seta lățimea maximă a unui element.

border - proprietățile marginii CSS vă permit să specificați stilul, lățimea și culoarea chenarului unui element.

margin - marginile sunt folosite pentru a crea spațiu în jurul elementelor, în afara oricăror granițe definite.

padding - padding este folosit pentru a crea spațiu în jurul conținutului unui element, în interiorul oricăror margini definite.
