Elemente si tag-uri

Elemente

Elementele in HTML reprezinta tot ceea ce este cuprins de la tag-ul de inceput pana la tag-ul de inchidere

Elementele in html sunt:
Definite de un tag de inceput, continut si tag final
Exemplu:
(exp:<p>continut</p>)
Fara continut
fara tag de inchidere - se inchid singure
exemplu:<br/>, nu are tag de inchidere
Nested - elementele pot contine alte elemente
Exemplu: <html> <body> <h1>My First Heading</h1> <p>My first paragraph.</p> </body> </html>

Tag-uri

Ce sunt tag-urile? Un tag este un cuvant sau o litera inconjurata de < si >

  1. Tag-ul html - elementul principal
    Tag-urile pentru html sunt <html>...</html>
    Toate paginile web incep cu acest element
    Tag-ul <html> se deschide la inceputul documentului html
    Tag-ul </html> inchide documentul html
    Tot continutul unei pagini web este cuprins intre aceste 2 tag-uri
  2. Tag-ul head - Sectiunea head a site-ului
    Tag-urile pentru head sunt <head>...</head>
    Contine informatii despre website
    Ce elemente puteti pune aici:
    Title - titlul paginii
    Link - folosit la adaugarea fisierelor .css si a favicon-ului
    Meta - folosit la specificarea setului de caractere, descrierea paginii,etc
    Script - folosit la adaugarea de cod JavaScript
  3. Tag-ul title - titlul paginii
    Tag-urile pentru title sunt: <title>...</title>
    Acest element contine titlul paginii
    Titlul este afisat in bara de titlu a browserului, bookmarks, rezultatele cautarilor, etc.
    Titlul trebuie sa descrie continutul paginii succint si precis
  4. Tag-ul body - "corpul"- continutul paginii
    Tag-urile pentru body sunt <body>...</body>
    Elementul body apare imediat dupa inchiderea tag-ului head
    In interiorul acestui tag se scrie tot continutul paginii web
  5. Tag-urile h - titlurile
    Tag-urile pentru titluri sunt: <h..>...</h..>
    Tag-urile h se numeroteaza de la 1 la 6, 1 fiind cel mai important
  6. Tag-ul p-paragraful
    Tag-ul pentru paragraf este <p>....</p>
    Este unul dintre cele mai utilizate tag-uri
    Acest element da posibilitatea de a creea pragrafe
  7. Tag-ul a - un link
    tag-ul pentru link este:<a href="link">...</a>
    Textul care urmeaza sa fie transformat in link se pune intre tag-urile a,iar adresa la care sa se trimita se pune in atributul href
    Iti permite sa creezi link-uri catre alte pagini(interne sau externe)
  8. Tag-ul img - o imagine
    Tag-ul pentru img este <img src="sursa imagine" alt="text alternativ">
    Pentru afisarea imaginii este necesar sa fie incarcata undeva-host de imagini, server, etc.
    Tag-ul se inchide singur, nu are tag de inchidere
  9. Tag-ul div - element de tip block
    Tag-urile pentru div sunt <div>...</div>
    Elementul div este un container generic
    Este folosit pentru a structura o pagina mai bine
    Se poate folosi pentru a grupa mai multe paragrafe sau titluri din aceasi categorie
    Elementele div sunt folosite pentru:
    • partea de sus/jos a website-ului
    • coloane de continut si meniuri laterale
    • casuta de text
    • zone de design
    • galerii
    Cu ajutorul claselor si id-urilor se pot adauga stiluri CSS
  10. Tag-ul span - container inline
    Tag-urile pentru span sunt <span>....</span>
    Este similar cu elementul div, insa span este un element inline comparativ cu div

-Elementele de tip block, precum div, h1 si p sunt elemente creeate pentru a tine un continut relativ mare precum un paragraf sau un text complet. Un element de tipul block este intotdeauna deschis pe o linie noua.

-Elementele inline, precum span, a, img, etc sunt creeate pentru a tine un continut mic, precum un cuvant sau o propozitie. Un element inline nu trebuie adaugat pe noua linie.Elementele de tip block pot contine elemente inline, dar nu se poate invers.
